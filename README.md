# Scheduling Automation App
 Smart Scheduling Automation App
Ek powerful Python-based tool jo Google Calendar API aur Pandas ka use karke aapke busy slots ko analyze karta hai aur real-time mein meetings book karne mein madad karta hai.

🚀 Features
Google Calendar Integration: Aapke primary calendar se real-time events fetch karta hai.

Busy/Free Analysis: Pandas DataFrame ka use karke gaps (free slots) calculate karta hai.

Automated Booking: User se input lekar automatically calendar mein event create karta hai aur guest ko invite bhejta hai.

Web Interface: Streamlit ka use karke ek saaf-suthra dashboard pradan karta hai.

🛠️ Tech Stack
Language: Python 3.x

Data Analysis: Pandas

API: Google Calendar API (OAuth 2.0)

Web Framework: Streamlit

Version Control: Git & GitHub

📋 Installation & Setup
Repository Clone Karein:

Bash

 code =   git clone https://github.com/Deepak-Yadav555/scheduling-automation-app.git
code =    cd scheduling-automation-app


Virtual Environment Banayein:

Bash



code =    python -m venv venv
code =    source venv/bin/activate  # Windows ke liye: venv\Scripts\activate



Dependencies Install Karein:

Bash


code =    pip install -r requirements.txt


Google API Credentials:

Google Cloud Console par jayein aur ek naya project banayein.

Google Calendar API enable karein aur OAuth 2.0 credentials download karke credentials.json naam se project folder mein rakhein.

App Chalaein:

Bash


code =    streamlit run app.py
